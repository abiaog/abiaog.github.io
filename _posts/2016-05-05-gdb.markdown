# Core Dump Example

[gdb_core_dump.c](/code_for_post/gdb/gdb_core_dump.c)   

	gcc -g code_for_post/gdb/gdb_core_dump.c 
	ulimit -c unlimited
	./a.out 
	gdb ./a.out core
